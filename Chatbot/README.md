# Laravel AI Chatbot (Local Mistral)

A simple AI chatbot built with **Laravel**, **Blade**, and **Alpine.js**, powered by a **local Mistral model** running through Ollama.

This project demonstrates how to build a lightweight ChatGPT-style interface using Laravel and connect it to a locally hosted LLM.

---

# Features

## Phase 1 (MVP)

- Basic chat UI
- Send messages
- Display AI responses

## Phase 2

- Chat history
- Multiple conversations
- Markdown rendering

## Phase 3 (Planned)

- Streaming responses
- File upload
- RAG knowledge base
